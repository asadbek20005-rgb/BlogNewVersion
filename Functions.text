 1.Publish / Unpublish
Maqolani yozgan odam admin/moderator tasdiqlamasa, draft holatda turadi.
Publish qilganda ommaga ko‘rsatiladi.

2. Like / Dislike (Reactionlar)
Har bir postga foydalanuvchi "like" yoki "dislike" bosishi mumkin.

 3. Comments / Replies
Foydalanuvchi postlarga izoh yozadi.

Izohlarga javob (reply) qilish ham qo‘shiladi (nested comments).

4. Search and Filter
Sarlavha, tag, muallif, sana bo‘yicha qidirish.

Masalan: ?search=aspnet, ?tag=blazor

5. Categorization / Tagging
Har bir post:

Category ga tegishli bo‘ladi (masalan: Programming, Life, Business).

Bir nechta Tag ga ega bo‘ladi (masalan: C#, ASP.NET, Blazor).

 6. Notification System
Yangi post chiqqanda foydalanuvchilarga bildirishnoma boradi.

Yoki kimdir izoh yozsa, post egasiga xabar boradi.

7. Views Count
Har bir post ochilganda view count ortadi.

Ko‘p o‘qilgan maqolalarni alohida chiqarish mumkin.

 8. Bookmark / Save for Later
Foydalanuvchi postni "saqlab qo‘yish" (Read Later) uchun belgilaydi.

 9. Share Functionality
Har bir postni boshqa foydalanuvchilarga ulashish (telegram, email, link nusxalash).

 10. Admin Approve / Reject
Blog yozuvlari admin/moderator tomonidan tekshiriladi.

Adminlar Approve, Reject yoki Edit qilishadi.
